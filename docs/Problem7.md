# Problem Statement

1) Creat an Animal class with the following attributes:
   * name
   * species
   * zoo
   * sound that it makes
2) Create a constructor and all getters and setters for the animal class

3) In a separate class (problem7.java), do the following:
   1) Make an ArrayList that contains three Animal objects of your choice
   2) use a loop to print each animal's name and species
   3) Modify one animal's zoo to a different zoo and print out the new one
   4) Print the sentence "_____ is a____ that lives in ______ and says _____" for one *randomly chosen* animal from your list.
